# Romlaw Advocates LLP Website

## Overview
A professional law firm website for Romlaw Advocates LLP, a full-service law firm based in Kenya. The website showcases their legal services, team, case studies, and provides contact information for potential clients.

## Project Structure
```
├── server.py          # Python HTTP server for development
├── src/               # Static website files
│   ├── css/
│   │   └── style.css  # Main stylesheet
│   ├── images/        # Local images
│   ├── includes/      # HTML partials (navbar, footer, top-bar)
│   ├── js/
│   │   └── script.js  # JavaScript functionality
│   ├── index.html     # Homepage
│   ├── blog.html      # Blog page
│   ├── case-studies.html  # Case studies page
│   └── practice-areas.html  # Practice areas page
└── replit.md          # This file
```

## Technology Stack
- **Frontend**: Static HTML, CSS, JavaScript
- **Server**: Python HTTP server (development)
- **Fonts**: Google Fonts (Cormorant Garamond, Montserrat)
- **Icons**: Font Awesome 6.0

## Development
The development server runs on port 5000 using a simple Python HTTP server with no-cache headers.

## Deployment
Configured as a static site deployment serving from the `src/` directory.
